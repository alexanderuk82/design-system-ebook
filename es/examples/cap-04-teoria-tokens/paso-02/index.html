Resolucion vs referencia

El mismo set de tokens compilado de dos maneras. En resuelto, cada variable lleva su valor literal. En referenciado, la cadena se preserva con var(). Las dos versiones funcionan, pero se comportan distinto cuando intentas cambiar el primitivo en runtime.

var(--color-brand-primary) usa el primitivo via la cadena

Mueve el color picker. En modo referenciado el cuadro cambia, porque el semantic apunta a var(--color-blue-500). En modo resuelto se queda igual, porque el semantic ya tiene el hex literal compilado dentro.
