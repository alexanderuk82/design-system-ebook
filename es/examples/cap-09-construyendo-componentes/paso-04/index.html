Select

El Select nativo es feo e inconsistente entre plataformas. Construido sobre Radix UI obtienes navegacion con teclado, type-to-jump, escape para cerrar y portal para escapar del overflow del padre, todo gratis. Tu trabajo es estilizar.

Abre con click o Enter. Navega con flechas. Cierra con Escape. Tipea una letra para saltar al primer item que coincide. Todo viene de Radix.
