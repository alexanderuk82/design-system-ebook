## When to use each variant

- **primary**, the main screen action. Only one per context.
- **secondary**, important but non primary actions. Cancel, back, and similar.
- **ghost**, support actions or secondary navigation.
- **danger**, irreversible actions with serious consequences.

## Accessibility

The Accessibility panel runs axe-core against every story and reports
violations, passes, and inconclusive results. If you swap the color tokens to
low contrast values, the affected story will light up here.

The Button respects `prefers-reduced-motion`. When the user has it on, the
loading spinner renders static instead of rotating.
